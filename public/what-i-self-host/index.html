What I Self-Host

I self-host several applications and websites on my home server:

  • Home Assistant for home automation
  • Authelia for authentication in front of my homemade web applications
  • Deluge for torrents
  • ddclient for dynamic DNS
  • ESPHome Dashboard for flashing smart home devices that run ESPHome
  • Linkding for bookmarks (like Pinboard)
  • Miniflux for RSS feeds (like Feedly)
  • My personal websites
  • Nextcloud for file storage (like Google Drive) and photo backup (like Google Photos)
  • Wallabag for read-it-later (like Pocket or Instapaper)
