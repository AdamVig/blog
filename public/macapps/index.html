My Favorite Mac Apps

  • InsomniaX: a feature-rich menubar app that allows you to suppress idle sleep and lid sleep on a laptop.
  • Skim: a fast and efficient PDF reader with annotation tools.
  • Display Menu: free with optional $2 pro version upgrade; allows you to adjust display resolution easily, useful to get more space on Retina Macbook displays.
  • Monity: $5, tasteful system monitoring app that displays statistics and info in the Notification Center.
  • Goofy: Facebook messenger app that is essentially a wrapper for Messenger.com but provides native OS X notifications for new messages.
  • Spectacle: window manager accessible through menubar or system-wide keyboard shortcuts, useful for quickly making windows fill the screen or go split screen.
  • Disk Inventory X: disk usage analysis tool to help you visualize what is taking up the most hard drive space
  • The Unarchiver: capable utility that can extract a wide variety of archive formats
  • iTerm 2: customizable replacement for Terminal.app
